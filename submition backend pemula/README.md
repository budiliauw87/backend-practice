# Submission Bookshelf API

#### Tugas membuat project bookshelf 5 kriteria

- API dapat menyimpan buku
- API dapat menampilkan seluruh buku
- API dapat menampilkan detail buku
- API dapat mengubah data buku
- API dapat menghapus buku

### Pengujian API
Pengujian API mengunakan POSTMAN Collection dan Environment sesuai dengan yang disedia di 
[link ini](https://github.com/dicodingacademy/a261-backend-pemula-labs/raw/099-shared-files/BookshelfAPITestCollectionAndEnvironment.zip)


### Installasi

Membutuhkan program [Node.js](https://nodejs.org/) v10+ 

```sh
cd to directory
npm run start
```
